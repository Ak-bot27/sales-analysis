Sales Analytics Dashboard
Overview

This project presents an interactive Sales Analytics Dashboard designed to analyze sales performance and support data-driven decision-making.
The dashboard provides key business insights through visualizations of revenue, profit, orders, customer segments, 
regional performance, and product categories.

Objectives
Monitor overall sales performance.
Track revenue, profit, and order trends.
Identify top-performing products and categories.
Analyze regional sales performance.
Understand customer purchasing patterns.
Support business decisions with interactive visualizations.

Dashboard Features
Executive KPI cards (Sales, total revenue,Orders)
Sales trend analysis
Regional sales performance
Category and sub-category analysis
Customer segment analysis
Top-performing products
Interactive filters and slicers for dynamic exploration

Tools & Technologies
Powerpoint Presentation
Microsoft Excel

Key Insights
Identified top-performing product categories.
Compared regional sales and profitability.
Analyzed customer segment contributions.
Tracked sales trends over time.
Highlighted business opportunities using KPI metrics.
